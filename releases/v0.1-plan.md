## Release Process

```bash
# 1. Build Chapters 1-3
knowledge-compiler build acsm12
# (run .prompt files through Reasonix CLI)
knowledge-compiler build acsm12 --stage generate
knowledge-compiler build acsm12 --stage decompose
knowledge-compiler validate acsm12

# 2. Tag release
knowledge-compiler release acsm12 v0.1 \
  --chapters 1 2 3 \
  --description "Exercise Science Fundamentals — PA benefits, screening, fitness testing"

# 3. Push to GitHub
git tag acsm12-v0.1
git push origin acsm12-v0.1
```
